# 修复CG跳过功能

## 1. 问题分析
当前CG播放使用的是`PlayVideo`脚本，而不是之前修改的`VideoManager`脚本。`PlayVideo`脚本直接附加在视频播放器上，负责播放视频和处理视频结束事件。因此，之前的修改没有生效。

## 2. 解决方案
修改`PlayVideo`脚本，添加按住P键3秒跳过CG的功能。

## 3. 具体修改

### 修改文件：`PlayVideo.cs`
1. 添加按住P键计时相关变量
2. 在Update()方法中添加按键监听和计时逻辑
3. 当按住P键3秒时，调用TurnScene()方法跳过CG

## 4. 修改细节
- 添加变量：`private float pKeyHoldTime = 0f`，用于记录按住P键的时间
- 添加常量：`private const float P_KEY_HOLD_DURATION = 3f`，定义按住时间阈值
- 在Update()方法中检查P键状态
- 按住P键时，累加时间；松开时，重置时间
- 当时间达到3秒且视频正在播放时，调用TurnScene()方法跳过CG

## 5. 预期效果
- CG播放时，按住P键3秒后，会直接跳转到下一个场景
- 松开P键时，计时会重置
- 方便调试，无需等待完整CG播放
