# 修改熬药环节为关卡特定药品

## 1. 问题分析
当前熬药环节使用随机药品机制，在 `CMD_DatabaseExtension_Medicine.CreateMedicinePot` 方法中，通过 `ItemWarehouse.Instance.GetRandomItemsExcluding()` 获取随机药品。用户要求将熬药环节出现的药品改为对应关卡的药品。

## 2. 解决方案
修改 `CreateMedicinePot` 方法，移除随机获取药品的逻辑，改为只使用目标药品或与关卡相关的药品。

## 3. 具体修改

### 修改文件：`CMD_DatabaseExtension_Medicine.cs`
1. 定位到 `CreateMedicinePot` 方法
2. 修改第69-73行的逻辑
3. 将随机获取药品的代码替换为固定逻辑
4. 确保熬药环节只出现当前关卡所需的药品

## 4. 修改细节
- 原代码：`string[] randomOtherMedicine = ItemWarehouse.Instance.GetRandomItemsExcluding(totalCount, targetMeds);`
- 修改后：移除随机逻辑，只使用目标药品或关卡特定药品
- 保持其他逻辑不变，确保熬药环节正常运行

## 5. 预期效果
熬药环节将不再出现随机药品，只会出现对应关卡所需的特定药品，与第一关保持一致。
